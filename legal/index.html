1. Data Storing
1a: Storing, Logging and using user-data:
  • Modtech will store your user ID and time of the last message sent for five minutes.
  • Modtech will store the amount of messages you've sent within 2 minutes.
1b Storing and/or using server data:
  • Modtech stores your servers ID, it is in no way associated with you or your members.
1c: Why we store this data:
  • We store your servers ID to provide you with custom prefixes through the 'setprefix' command and to keep track of server settings, modlogs and support tickets. User IDs are stored for automod detection only.
1d: Removal of data:
  • All server data is deleted upon removal of the bot or manual reset via rerunning the command that you used to set up (eg settings). User data is stored for five minutes, where it will then be deleted.
  • Alternatively you can contact any modtech developer, and they should remove it as soon as they can.
1e: Who can access the data:
  • Only trusted developers that have been granted access.
